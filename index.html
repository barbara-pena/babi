$ git clone [comando para clonar os projetos]
$ cd [comando para entrar na pasta do projeto]
$ git config --global user.name "" [comando para adicionar suas credenciais - nome]
$ git config --global user.email "seu_e-mail" [comando para adicionar suas credenciais - email]
$ git add . [comando para adicionar as alterações]
$ git commit -m "comentário sobre as alteracoes feitas" [comando confirmar as alterações feitas]
$ git pull [comando para puxar a última versao do código]
$ git push -u [comando para empurrar sua branch para o repositório remoto] / OU / $ git push --set-upstream origin [somente no primeiro commit]
